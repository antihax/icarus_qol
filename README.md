# Icarus QoL .pak Install Instructions
Quality of life changes which halfs most crafting requirements, doubles output of some items, increases generation of water and power and decreases consumption.


## Install (Server & Client)

1. Download the latest release files from GitHub:
  - icarus_qol.pak
2. Open your Icarus install folder.
3. Go to:
  - Icarus\Content\Paks
4. Make a folder called Mods if it does not exist
5. Copy icarus_qol.pak into that Mods folder.
6. Start (or restart) the game.

## Update

1. Download the newest icarus_qol.pak from the latest release.
2. Replace the old file in Icarus\Content\Paks\Mods.
3. Launch the game.

## Uninstall

1. Close the game.
2. Delete icarus_qol.pak from Icarus\Content\Paks\Mods.
3. Start the game.
